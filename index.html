Welcome to My Static Website!

This is a simple static website to learn deploying on Coolify.

About This Project

This project is a basic example of a static website. It includes HTML, CSS, and optional JavaScript.

Get Started

Follow the steps below to deploy this website on Coolify:

  1. Create a repository on GitHub and upload these files.
  2. Sign up on Coolify and connect your GitHub account.
  3. Create a new project on Coolify and select your repository.
  4. Deploy your website!
